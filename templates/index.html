Dataset

Jigsaw Toxic Comment Classification Challenge. (2017). Kaggle Competition Dataset. Contains toxic comment labels including toxic, severe_toxic, obscene, threat, insult, and identity_hate categories.

Bad Words Lists

Carnegie Mellon University Bad Words List. CMU Bad Words Resource. Comprehensive list of profane words for text filtering.

English Profanity Dataset

LDNOOBW. (2021). List of Dirty, Naughty, Obscene, and Otherwise Bad Words. GitHub Repository. Multi-language profanity detection wordlist.

Hinglish Profanity Classification

Mathur, P. (2021). Hinglish Offensive Text Classification. GitHub Repository. Research on offensive text detection in Hinglish language.

Contractions Dictionary

Andrew Bury. Contractions Dictionary. GitHub Repository. JSON mapping of English contractions for text normalization.

Natural Language Toolkit (NLTK)

Bird, S., Klein, E., & Loper, E. (2009). Natural Language Processing with Python. O'Reilly Media Inc. NLTK Official Website. Used for tokenization, lemmatization, and stopword removal.

Scikit-learn Machine Learning Library

Pedregosa, F., Varoquaux, G., Gramfort, A., et al. (2011). Scikit-learn: Machine Learning in Python. Journal of Machine Learning Research, 12, 2825-2830. Scikit-learn Documentation. Used for TF-IDF vectorization and Naive Bayes classification.

FastAPI Web Framework

Ramirez, S. (2018). FastAPI: Modern, fast (high-performance) web framework for building APIs with Python 3.7+. FastAPI Documentation. Used for creating the web application interface.

Research Methodology

Text preprocessing pipeline includes: URL removal, HTML tag cleaning, contraction expansion, tokenization with NLTK, stopword filtering, and lemmatization. Data augmentation performed using curated bad words from multiple sources to improve toxic content detection accuracy.
